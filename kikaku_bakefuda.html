化け札の夜（仮）

色の物に化けて隠れ、八尺様に見つからないよう全色そろえて鳥居から脱出する、
和ホラー版・非対称かくれんぼ（プロップハント）。
非対称マルチプロップかくれんぼ和ホラーUGC拡張配信映え

ひとことで

逃げる複数プレイヤー 対 八尺様になった1人のかくれんぼ。逃げる側は色付きの物（提灯・暖簾など）に化けて、同じ色の店に紛れて隠れる。八尺様は徘徊して探し、見つめて震わせ「ぽぽぽ」で追い詰める。お札で自分の色を変えながら全色をそろえ、鳥居をくぐって脱出すれば勝ち。

色の商店（隠れる舞台）

赤の店 青の店 緑の店 黄の店 紫の店

商店街には色ごとにまとまった店がある（赤い物だらけの店／青い店…）。自分の今の色と同じ店の同色の物に紛れて静止すれば本物と見分けがつかない。違う色の店にいると浮いてバレる。＝色を部屋単位でまとめるのでレベルも作りやすい。既存の5色プロップ資産をそのまま流用。

コアの遊び

🏮 逃げる側

  1. 自分の色の店に化けて隠れる
  2. 八尺様の隙にこっそり移動
  3. お札を拾う → 色が変わる（今の店が敵地に！）
  4. 次の色の店へ引っ越し…を繰り返す
  5. 全色そろえたら鳥居から脱出＝クリア

👺 八尺様（1人が操作）

  1. 商店街を徘徊して探す
  2. 怪しい物を見つめる → 相手が震え出す
  3. 震えてる隙に「ぽぽぽ」で削る
  4. 逃げたら追う（チェイス）
  5. 3発分削れば神隠し（流血なし）

中核メカニクス

① 恐怖ゲージ → 震え（駆け引きの芯）

全員に見えない恐怖ゲージがある。見つめられる・ぽぽぽを浴びると上がる。正しく隠れてる人は上がるのが遅い（粘れる）／違う色・移動中は速い（すぐ割れる）。ゲージが満タンでぶるぶる震える → そこから被弾する。
重要：震えは"元に戻せる"。視線を切って隠れ直せば恐怖が下がり震えが止まる＝「震えてる数秒でどう逃げ切るか」が中間の攻防。

② ぽぽぽ（連打OK・3発でアウト）

八尺様の主武器。連打していい。ただしダメージが入るのは相手が震えている隙のときだけ。震えるまでは恐怖を溜める役。誰も無敵ではない——うまく隠れた人も、張り付かれ続ければ最後は割れる。連打が「じわじわ追い詰める気持ちよさ」に。コヤッキーの「ぽ…ぽ…」が商店街に響く。

③ お札で色替え → 全色そろえて脱出

お札を取ると自分の色が変わる。変わった瞬間、今いた店が敵地になり別の色の店へ強制引っ越し。＝全色の店を渡り歩く必要があり「隠れっぱなし」は不可能。全色そろえたら鳥居をくぐって脱出。

フェアにする4つの約束

無敵ゼロ
完全に安全な状態は作らない。うまく隠れれば"粘れる"だけ。
震えは可逆
震えても視線を切れば回復。即死＝理不尽を排除。
回復は軽く1回
体力は少し・1試合1回だけ回復。籠って回復待ちで試合が止まるのを防ぐ。
粘着を無意味に
勝ちは「時間内に何人脱出させたか」のスコア制＋鬼に索敵1つ（耳澄まし）。1人固執を防ぐ。

わかりやすさ（ライト層＝視聴者向け）

恐怖ゲージは数値で出さず体感3段階で見せる：低＝正常／中＝心音＋画面端ノイズ／高＝手ブレ・息切れ（震え予兆）／MAX＝震え・被弾可。お札取得後は画面中央に「青の店へ！」と指示＋現在色アイコン＋光で誘導。推理させず、迷子にさせない。

UGC（増やしたい要素・段階導入）

Lv1（まずここ）：化けプロップの柄・模様だけ投稿（形は共通の規格）。画像審査だけで安全・当たり判定も壊れない。投稿者名を刻印＋コスメ販売。
Lv2以降：規格テンプレ内でのメッシュ投稿→公式キュレーション採用。「自由投稿→即採用」はやらない（審査・荒らし・負荷で破綻するため）。

配信映え

①1人が八尺様＝配信者が視聴者を狩る（鉄板フォーマット）。②震え → バレる瞬間の手に汗。③八尺様が「今の動いた!? 気のせい!?」と確信が持てない余白＝実況が一番盛り上がる。④終盤の鳥居前ダッシュ。⑤変な化けプロップ（UGC）で笑い。短い試合（5〜8分）で見せ場が何度も。

使える既存資産（＝作るのが速い）

✔ 和風プロップ117種（5色系）＋お札　／　✔ 迷路商店街の自動生成器（大通り・裏道・鳥居・巡回点）　／　✔ 八尺様モデル＋AI（視界・聴覚・徘徊）　／　✔ ブラウザ操作プロト（歩行・一人称/三人称・人間⇄八尺様切替）。
新規で作る核は「化け」「恐怖ゲージ→震え」「ぽぽぽ」の3つに集約。土台は8割そろっている。

作り方：まず"ちびテスト"

いきなり全部作らない。まず一番小さい形で「面白いか」だけ確かめる：

部屋1つ・色1つ・逃げる人1人・八尺様1人だけで、
「見られる → 恐怖たまる → 震える → ぽぽぽ → 視線を切って隠れ直す」——この1往復が楽しいかを検証。

面白ければ → 3色・4対1・お札・迷路生成・UGCへ拡張。面白くなければ数値でなく設計に戻る（安く済む）。※八尺様らしさの演出はコアが面白いと分かってからの味付け。

初期パラメータ（検証用のたたき台）

項目初期値
震え発動（恐怖の閾値）100
正しい色で見られる+8〜10 / 秒（粘れる）
違う色で見られる+25 / 秒（約4秒で震え）
移動中に見られる+20 / 秒
ぽぽぽ命中（震え前）+20〜25 / 発
震え継続（この間だけ被弾）3秒
ライフ3（被弾後1.5秒無敵）
ぽぽぽ連射間隔1.2秒
視線を切って隠れた時の減衰−10 / 秒
ライフ回復同色店で8〜10秒無事＝1回復・1試合1回
推奨マッチ逃げ4〜6 ＋ 鬼1 ／ 1試合5〜8分
